# demo-rest-api

## keywords: ES6 - Babel - Node.js - Express - REST API - Jest - Docker

### Installation

```sh
$ git clone https://github.com/Adarsh236/demo-rest-api.git my-project
$ cd my-project
$ npm i
$ npm run dev
```

### Build and production

```sh
$ npm run build
$ npm run start
```

### Jest testing

```sh
$ npm run test
// or
$ npm run test-watch
```

### For Docker

```sh
$ docker-compose build
$ docker-compose up
```
